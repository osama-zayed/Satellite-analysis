Endpoint

/api/add/image

Method

POST

Parameters

  • image (Required): The image file to be processed. (Type: File)
  • model-select (Required): The selected model. It can have one of the following values:
    • buildingDensity
    • buildingDetector
    • routeGenerator
    • slum_detection

Response Format

Success (Status Code: 200)

{
"state": "200"
"message": "Successfully",
"image": "/static/image/output_image.png",
}

Failure (Status Code: 500)

{
"state": "500"
"message": "Error",
}
